# Tech Stack

本项目使用的技术栈：

- **框架：** Next.js (React)
- **语言：** TypeScript
- **样式：** Tailwind CSS

---

**开发注意事项：**

- 组件使用 `.tsx` 扩展名
- 样式优先使用 Tailwind 工具类
- 类型定义放在 `types/` 目录或使用 `.d.ts` 文件
- 遵循 Next.js App Router 或 Pages Router 约定（根据项目结构）

---

**设计规范：**

- ❌ **避免蓝紫渐变色** (blue-purple gradients)
- ✅ 推荐使用：单色系、暖色渐变、冷色渐变（非蓝紫）、中性色
